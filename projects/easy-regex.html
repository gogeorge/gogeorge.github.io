EasyRegex is a Javascipt library that makes it easier to use regular expressions.

Regular Expressions are complex sequences of symbols and characters that are used to find and search somethign in a string. Regular expressions can check if a string contains numbers or a specific word/phrase etc.

The library is consisted of several functions, each with appropriate and easy to remember names, that return their regular expression. The functions can be combined thus making it easier for the user to achieve what they want.

Here is an example:

Let's say you want to check if a string contains gmail or yahoo email addresses but not any greek addresses which end in '.gr'. In other words, you want to check if the string contains '@gmail' OR '@yahoo' but NOT '.gr'. Let the string be var a.

So without using any libraries this is a possible way to check the string: 


	var a = "John has this email: example@gmail.com and Doe has this one example@yahoo.com etc...";

	if (a.match('.*\b@gmail\b.*') || (a.match('.*\b@yahoo\b.*') && a.match('.*^(?!.*.gr).*$'))) {
  		console.log("string contains either '@gmail' or '@yahoo' and not '.gr'");
	} else {
  		console.log('did not match');
	}
	/*
	The string contains everything that is needed and it doesn't contain
	a greek email therefore the string matches the regular expressions
	*/

The regular expressions on the above code block look confusing and complex when you can use this library to achieve something less confusing and more 'readable':



	var a = "John has this email: example@gmail.com and Doe has this one example@yahoo.com etc...";

	if (a.match(hasString('@gmail') + or() + hasString('@yahoo') + notString('.gr'))) {
  		console.log("string contains either '@gmail' or '@yahoo' and not '.gr'");
	} else {
  		console.log('did not match');
	}

In this example, these functions are used instead of regular expressions to achieve the same result.Each function returns a specfic regular expression. For example hasString('@gmail') will check if var a contains '@gmail' and notString('.gr') will check if var a does not contain '.gr'.

For more information visit the github repository where it also has a table of the rest functions that can be used.
