Yveete is an Intelligent Personal Assistant (IPA) that can perform various internet tasks to help the user such as creating a calendar event, to multitasking and sending emails. Yveete was developed using only JavaScript.

One day, I didn't now what to do with what I learned from codecademy so I thought of this idea: something like Siri, but with more tasks/commands and fewer conversations. Back then there weren't any digital assistants on computers like now Siri on macOS 10, and Cortana on Windows 10 so I had an advantage of having the program on a computer instead of a smartphone.

Yveete is now on github, https://github.com/gogeorge/Yveete, under the GPL 3.0 License

How it works,

Yveete's JavaScript is mainly if/else and else if statements. For example this is how to make a very simple interaction where the user types "hi" and Yveete replies with "hey":

if (question == "hi") {
    answer = "hey";
}

Now to add a voice you can do this:

if (question == "hi") {
    var msg = new SpeechSynthesisUtterance('hey');
    window.speechSynthesis.speak(msg);
    answer = "hey";
}

The code can be changed so the user can type "hi" and "hello":

if (question.toUpperCase() == "HI" || question.toUpperCase() == "HELLO") {
    var msg = new SpeechSynthesisUtterance('hey');
    window.speechSynthesis.speak(msg);
    answer = "hey";
}

Here is a more complex if/else statement where the user types their name and Yveete remembers it:

if (question.match(".*\\bmy name is\\b.*")) {
   //if what the user types matches the string 'my name is' then:
   var userName = question.substring(11);
   //save the name by extracting the substring after the 11th character
   if (userName.match(/[0-9]/i)) {
      //if the user typed only numbers or included a number in their name then:
       var msg1 = new SpeechSynthesisUtterance('Please type your name in letters only');
       window.speechSynthesis.speak(msg1);
       answer = "Please type your name in letters only";
   }
   else if ((userName != null) && (userName.match(/[a-z]/i))) {
     // if the user typed a name and it is in letters then:
      var msg2 = new SpeechSynthesisUtterance('I will remember your name as ' +       userName);
      window.speechSynthesis.speak(msg2);
      localStorage.setItem('userName', userName);
      answer = "I will remember you as " + userName;
   }
   else if (userName == null) {
      answer = "You didn't type your name";
   }
}
