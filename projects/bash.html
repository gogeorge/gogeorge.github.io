People use the command lines so that they have a better control over their computers. A bash profile makes it easier for people to use the Terminal app and the Bash shell. By creating a bash profile you customize the command line and give easier names to commands. As you can see from the animated command line above, this bash profile simplifies the commands and makes them easier to remember for example instead of ls -l there you can use list.Ideally instead of a long command such as python -m SimpleHTTPServer, you can use localhost.

You can find this bash profile on https://github.com/gogeorge/bash-profile and it has no license.

Installation,

*This can only be installed If you have a platform which runs on macOS like a macbook/iMac or Linux
*This profile works best on a black background and green text theme. You can customize your theme by going to Terminal > Preferences

1.) open terminal and type the following:

nano ~/.bash_profile

2.) Then copy the code from the Github file named 'source.bash' and paste it in the nano editor (in the command line)

3.) press ctrl + o (to save the file), then enter and then ctrl + x to exit the nano editor

4.) To enable the profile type the following:

source ~/.bash_profile
